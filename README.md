# EBAC Data Science Exercises

Repositório com projetos práticos desenvolvidos durante minha formação em Ciência de Dados na EBAC.

## Projetos em destaque

- **Classificação de Credit Score com Árvore de Decisão**
  - Tratamento e balanceamento de dados, avaliação com acurácia, precisão, recall, F1-score e matriz de confusão.
  - Tecnologias: Python, Pandas, Scikit-learn

- **Dashboard de E-commerce com SQL e Power BI**
  - Consolidação de dados de transações e clientes com SQL e construção de dashboard com indicadores de vendas e perfil de clientes.
  - Tecnologias: SQL, Power BI

- **Regressão Linear para previsão de valor de aluguel**
  - Análise exploratória, correlação e comparação entre regressão simples e múltipla.
  - Tecnologias: Python, Pandas, Scikit-learn

- **Pré-processamento e análise de dados para score de crédito**
  - Limpeza, imputação, padronização, correlação, encoding e balanceamento com SMOTE.
  - Tecnologias: Python, Pandas, Scikit-learn
