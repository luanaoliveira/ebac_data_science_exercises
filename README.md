# EBAC Data Science Exercises

Repositório com projetos práticos desenvolvidos durante minha formação em **Ciência de Dados na EBAC**, com foco em **Python, SQL, análise exploratória de dados, modelagem preditiva, visualização de dados e Power BI**.

## Projetos em destaque

### Classificação de Credit Score com Árvore de Decisão
Projeto de classificação para análise de score de crédito, com tratamento e balanceamento de dados, separação entre treino e teste e avaliação com métricas como acurácia, precisão, recall, F1-score e matriz de confusão.  
**Tecnologias:** Python, Pandas, Scikit-learn

### Dashboard de E-commerce com SQL e Power BI
Projeto voltado à análise de vendas e perfil de clientes, com consolidação de dados em SQL e construção de dashboard no Power BI com indicadores como total de vendas, ticket médio, clientes únicos e vendas por categoria e estado.  
**Tecnologias:** SQL, Power BI

### Regressão Linear para previsão de valor de aluguel
Projeto de modelagem preditiva com análise exploratória, correlação e comparação entre regressão linear simples e múltipla para previsão do valor de aluguel.  
**Tecnologias:** Python, Pandas, Scikit-learn

### Pré-processamento e análise de dados para score de crédito
Projeto com foco em limpeza, imputação, padronização, correlação, codificação de variáveis e balanceamento com SMOTE para preparação da base para modelagem.  
**Tecnologias:** Python, Pandas, Scikit-learn
